# cpu-monitor

Маленькая утилитка для просмотра нагрузки/температуры ЦП через значок трея. 
Есть возможность менять цвета =)


### Python 3.9

### Запуск через `pip`

    pip install -r requirements.txt
    python main.py

### Запуск через `poetry`

    poetry shell
    poetry install
    poetry run python main.py
